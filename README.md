🍊 Fanta Digital Experience

Este projeto é um estudo de interface focado em UI/UX moderna e interatividade dinâmica. O objetivo foi transformar um catálogo comum de bebidas em uma jornada visual vibrante, utilizando as cores icônicas da Fanta com uma estética tecnológica.
🚀 O que foi testado?

O foco principal deste projeto não foi a estrutura final de um site completo, mas sim a experimentação de três pilares:

    Estética Glassmorphism: Aplicação de camadas de transparência e desfoque (blur) para criar profundidade.

    Lógica de Slides com JS Puro: Desenvolvimento de um slider interativo onde a transição de imagens é acompanhada por mudanças cromáticas no layout.

    Identidade Visual: Uso da tipografia Bebas Neue e elementos 3D flutuantes para reforçar o aspecto "divertido e tecnológico".

🛠️ Tecnologias Utilizadas

    HTML5 (Semântica base)

    CSS3 (Grid, Flexbox, Glassmorphism e Keyframes para os elementos 3D)

    JavaScript (ES6+) (Manipulação de DOM para controle do Slider)

⚠️ Observação Importante

    Status do Projeto: Este site foi desenvolvido como um teste de design e lógica de programação.

        Responsividade: O projeto ainda não possui responsividade. Ele foi otimizado exclusivamente para visualização em resoluções Desktop.

        Foco: O intuito foi validar as transições de cores e a fluidez dos scripts de animação.
